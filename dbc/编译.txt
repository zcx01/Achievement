1、安装dock 
安装ubuntu18
参考
https://docs.google.com/document/d/16m0knCqTYxRU9zGxcH_U8rSCfpCVTrBVb9kI93T4UJA/edit

2、进入dock
docker exec -it kiara_build_18 bash

3、进入编译目录（使用工具编译）
cd ~/Works/Repos/tool_parser

4、执行 make menuconfig
选中Load an Alternate Configuration file
在输入框中输入VendorFiles/config/.config
保存退出

5.执行sudo make

6.把output文件拷贝到~/Works/Repos/changan_c385/src/ic_service/parser/generated下

(注*) 要在~/Works/Repos/tool_parser目录下创建软件链接VendorFiles指到~/Works/Repos/changan_c385/src/ic_service/parser/VendorFiles
